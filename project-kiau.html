Kiau Tech Website

Status: In Progress  •  Stack: HTML • CSS • JavaScript

View Live Site ↗

Overview

An ongoing solo project building and evolving the public-facing website for Kiau Tech — a regenerative agriculture company based in Colorado. I joined as a solo web developer in November 2024 and have been responsible for the full design and development of the site ever since.

One of the most interesting parts of this project is watching how much it has changed — in design philosophy, technical ambition, and scope — over a relatively short time.

Evolution & Timeline

Late 2024 — v1: The Beginning

When I joined, the site already existed but was very barebones. The signature feature from the start was a node-based navigation graph — an interactive web of nodes representing site sections. I built this myself from scratch. At this stage it was simple: just a handful of nodes, minimal styling, and a basic landing page.

HTML • CSS • JavaScript

2025 — v2: Growth & New Pages

Over the course of 2025 the site grew considerably. New pages were introduced to cover the company's mission, projects, and grant applications. The node graph was significantly improved — more nodes, better layout, smoother interactions, and a more polished visual style. This became the main navigation metaphor for the site.

HTML • CSS • JavaScript • D3 • API Integration

Late 2025 / 2026 — v3: The Archive

Eventually the node graph transitioned from a navigation tool to an archive page — a living showcase of all Kiau Tech's projects and initiatives. The main navigation moved to a more traditional structure, allowing the graph to shine as a visual portfolio of the company's work. The site continues to evolve as Kiau Tech grows.

HTML • CSS • JavaScript • Data Visualization

What I Learned

  • Working solo on a real client project with shifting requirements
  • Building custom interactive data visualizations from scratch (the node graph)
  • Coordinating with other departments to represent the company accurately in grant applications
  • Iterating on a live production website without breaking things
  • Balancing creative design with practical usability
