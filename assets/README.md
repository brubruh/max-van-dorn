# Resume Assets

This folder contains downloadable assets for the portfolio website.

## Adding Your Resume

1. **Create your resume** as a PDF file
2. **Name it consistently**: `Maxim_Van_Dorn_Resume.pdf` 
3. **Place it in this folder**
4. **Update all resume links** to point to: `assets/Maxim_Van_Dorn_Resume.pdf`

## File Naming Best Practices

- Use underscores instead of spaces
- Include your full name
- Keep the filename professional
- Use .pdf format for best compatibility

## Example Files

- `Maxim_Van_Dorn_Resume.pdf` - Your main resume
- `Maxim_Van_Dorn_Resume_2024.pdf` - Date-specific version
- `Maxim_Van_Dorn_Cover_Letter.pdf` - Optional cover letter template

## Security Note

These files will be publicly accessible once uploaded to GitHub, so only include information you're comfortable sharing publicly.
