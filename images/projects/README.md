# Project Images

This folder contains screenshots and images for the portfolio projects. 

## Image Guidelines

- **Format**: Use JPG for photographs, PNG for graphics with transparency
- **Size**: Recommended dimensions of 800x400px (2:1 aspect ratio)
- **File naming**: Use kebab-case matching the project title
- **Optimization**: Compress images to keep file sizes under 500KB

## Current Images

- `kiau-tech.jpg` - Screenshot of the Kiau Tech website
- `portfolio.jpg` - Screenshot of this portfolio website

## Adding New Project Images

1. Add your image file to this folder
2. Update the corresponding project entry in `../data/projects.json`
3. Use the relative path: `images/projects/your-image.jpg`
