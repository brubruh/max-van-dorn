NBA Props Betting Tool

Status: In Progress  •  Stack: Python • NBA API • Machine Learning • Data Analysis

Private Repository

Overview

A personal machine learning project that pulls live and historical data from the NBA's official API, analyzes player performance against specific matchups, and compares model predictions to current betting lines — surfacing the best over/under opportunities on the market today.

How It Works

1. Data Ingestion

Pulls player stats, game logs, and opponent defensive ratings directly from the NBA API. Data is cleaned and structured for analysis.

2. Matchup Analysis

For each player prop (points, rebounds, assists, etc.), the model compares historical performance in similar matchups — factoring in the opposing team's defensive tendencies, pace, and positional matchup data.

3. Prediction

A machine learning model generates predicted stat lines based on the historical matchup profile. These predictions are compared against the player's current market lines.

4. Line Comparison & Edge Detection

The tool scans current betting markets and flags props where the model's prediction diverges significantly from the line — highlighting potential over or under values to act on.

Key Challenges

  • Cleaning and normalizing inconsistent API data across seasons
  • Building matchup-specific features (not just raw averages)
  • Keeping predictions sharp without overfitting to small sample sizes
  • Automating fresh predictions each day before lines move
